# DaniiKodit — портфолио

Тёмный пиксельно-дизеринговый лендинг: циклическая пиксельная волна на canvas,
генеративные обложки проектов, переключение RU/EN, отдельная страница портфолио.

Чистые HTML/CSS/JS, без сборщиков и фреймворков — открывается как есть.

## Локальный запуск

```
python -m http.server 8123
```

и открыть http://localhost:8123

## Структура

- `index.html` — главная страница
- `portfolio.html` — подробное портфолио
- `css/style.css` — дизайн-система и стили
- `js/main.js` — переводы RU/EN, дизеринг-canvas, анимации
