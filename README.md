# Крис Мазур — портфолио

Тёмный пиксельно-дизеринговый лендинг: живой дизеринг-фон на canvas,
генеративные обложки проектов, переключение RU/EN.

Чистые HTML/CSS/JS, без сборщиков и фреймворков — открывается как есть.

## Локальный запуск

```
python -m http.server 8123
```

и открыть http://localhost:8123

## Структура

- `index.html` — вся разметка
- `css/style.css` — дизайн-система и стили
- `js/main.js` — переводы RU/EN, дизеринг-canvas, анимации
